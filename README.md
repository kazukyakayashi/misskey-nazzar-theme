# misskey-nazzar-theme
Theme for Misskey

# Use

Copy this code on misskey theme editor :

## Dark theme

```
{
	id: 'Nazzar-dark',

	name: 'Nazzar-dark',
	author: 'KazukyAkayashi',
	desc: 'Dark theme',

	base: 'dark',

	props: {
	accent: '#bf616a',
	bg: '#2b303b',
	fg: '#e5e7eb',
	fgHighlighted: 'rgb(163, 190, 140)',
	panel: '#2f3541',
	header: 'rgba(47, 53, 65, 0.75)',
	link: '@accent',
	mention: '#b48ead',
	hashtag: '#ebcb8b',
	renote: '#a3be8c',
	pageBg: '@bg',
	badge:'#bf616a',
	divider:'rgba(43, 48, 59, 0.7)',
	driveFolderBg:'@panel',
	scrollbarHandleHover:'@accent',
	},
}
```
